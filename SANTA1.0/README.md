## Description
A logistics management system built with Java, Spring Boot, MySQL, and OpenAPI.

## Features
- Product management
- Order tracking
- REST API with Swagger UI(Option)
- Database integration with MySQL

## Prerequisites
- Java 17
- MySQL 8
- Gradle
